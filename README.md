## 📌 Overview

The **BM & LO Productivity MIS Dashboard** is an interactive Microsoft Excel dashboard developed to monitor and analyze the performance of Loan Officers (LOs) and Branch Managers (BMs).

The dashboard provides real-time business insights through KPI cards, dynamic slicers, PivotTables, and PivotCharts, helping management make data-driven decisions quickly.

---

## ✨ Features

- 📈 Interactive KPI Cards
- 🎯 Collection Target vs Achievement
- 💰 MTD Collection Analysis
- 👥 Total Clients & OD Clients
- 🏦 Outstanding Collection Analysis
- 📍 State-wise Collection Analysis
- 🏢 Funder-wise Collection Distribution
- 👨‍💼 LO Attendance Distribution
- 🔝 Top 10 Performing Branches
- 🎛 Dynamic Excel Slicers
- 📊 Pivot Charts & Pivot Tables
- ⚡ GETPIVOTDATA-based KPI Automation
