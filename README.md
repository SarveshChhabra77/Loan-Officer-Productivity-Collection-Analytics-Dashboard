# BM & LO Productivity MIS Dashboard

## Overview

Interactive Excel MIS Dashboard developed for monitoring Loan Officer (LO) and Branch Manager (BM) productivity.

## Features

- KPI Cards
- Dynamic Slicers
- Pivot Charts
- Attendance Analysis
- Collection Analysis
- Outstanding Analysis
- Target vs Collection
- Top 10 Branches
- State-wise Performance
- Funder-wise Collection

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- GETPIVOTDATA
- Dashboard Design

## Skills Demonstrated

- Data Analysis
- Dashboard Development
- Business Reporting
- MIS Reporting
- Data Visualization
